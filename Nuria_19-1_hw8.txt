Изучить SQL запросы - CREATE, INSERT, SELECT, UPDATE, DROP
так же создать файл docs.txt и написать что делает каждый из этих методов

CREATE DATABASE
Команда для создания новой базы данных

INSERT
Команда INSERT INTO <table_name> в SQL отвечает за добавление данных в таблицу

SELECT
SQL-команд SELECT для получения данных из выбранной таблицы

UPDATE
SQL-команда для обновления данных таблицы

DROP DATABASE
Стандартная SQL-команда для удаления целой базы данных
